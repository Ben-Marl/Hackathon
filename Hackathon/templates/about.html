About Sharp Portfolio

This project is a web implementation of Harry Markowitz's Portfolio Theory. The tool fetches historical stock data from Yahoo! Finance and uses satistical methods of mean and variance to calculate the most optimal risk-adjusted portfolio distribution that lies on the efficient frontier.

The application is built with Flask framework for the backend and uses basic HTML, CSS & Bootstrap for the front end. Data is stored in a MongoDB database. Project implementation can be found at GitHub.

What's with the name?

To sharpen your portfolio, the algorithm behind this application generates 10000 different portfolio distributions and then picks the one from the efficient frontier which has the maximum Sharpe Ratio - a performance measure for investments developed by William F. Sharpe.
Hence the name Sharp Portfolio!
